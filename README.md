# TypeScript Interfaces

This repository contains my TypeScript practice code for learning Interfaces and related concepts.

## Topics Covered

* Interfaces
* Object Structure (Blueprint)
* Object Iteration
* Declaration Merging
* Optional Properties
* Interface Inheritance using `extends`

## Concepts

### Interface

Used to define the structure of an object.

### Declaration Merging

Multiple interfaces with the same name are automatically merged.

### Extends

Allows one interface to inherit properties from another interface.

## Files

* `app.ts` - TypeScript source code
* `app.js` - Compiled JavaScript code

This repository is part of my TypeScript learning journey and contains examples for practicing interfaces.
